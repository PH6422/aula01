# 🧮 Calculadora em Python com Histórico

Este projeto é uma **calculadora simples em Python** executada no terminal.  
Ela permite realizar operações matemáticas básicas e visualizar o **histórico das operações realizadas** durante a execução do programa.

---

## 📋 Funcionalidades

- Realizar operações matemáticas básicas:
  - Soma (`+`)
  - Subtração (`-`)
  - Multiplicação (`*`)
  - Divisão (`/`)
- Armazenar o histórico das operações realizadas
- Visualizar o histórico de cálculos
- Encerrar o programa através do menu

---

## ⚙️ Tecnologias utilizadas

- **Python 3**
- Estrutura de repetição `while`
- Estrutura de decisão `match case`
- Listas (`list`)
- Entrada de dados com `input()`

---

## 📂 Estrutura do Código

O programa utiliza um **menu interativo no terminal** com três opções:

| Opção | Função |
|------|------|
| 1 | Realizar um cálculo |
| 2 | Mostrar histórico |
| 3 | Finalizar programa |

As operações realizadas são armazenadas em uma lista chamada `lista`.

Exemplo de histórico:


[5, '+', 3, '=', 8, '||', 10, '*', 2, '=', 20, '||']
